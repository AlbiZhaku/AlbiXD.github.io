Chip-8 Emulator

Chip-8 is a simple interpreted computer system from the 1970s used to run small games like Pong and Space Invaders. To better understand how computers execute instructions, I built my own Chip-8 emulator from scratch.

This project was a way for me to go beyond high-level programming and see what happens “under the hood.” Writing an emulator means pretending to be the CPU: reading opcodes, decoding them, and executing instructions exactly like real hardware would. GitHub
